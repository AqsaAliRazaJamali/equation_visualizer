# ✨ Math equation_visualizer

## 🚀 Live Demo

 (https://aqsaalirazajamali.github.io/equation_visualizer/)
 

I built a math visualizer from scratch — no libraries, no frameworks.  
Just pure HTML, CSS, and JavaScript.

Most people think math is boring. I used to think that too.  
Then I realized: math was never boring, the way it was shown was.

So I built something that changes that.


## 🌟 Features

- 24+ animated mathematical visualizations (continuously expanding)
- Real-time drawing of equations from scratch
- Smooth animations using HTML5 Canvas
- Designed for learning through visualization

---

##  Visual Highlights

🌀 Fibonacci Spiral — grows square by square, revealing the golden ratio (φ)  
❤️ Heart Curve — draws itself with a glowing animated trail  
🐚 Logarithmic Spiral — found in shells, hurricanes, and galaxies  
✨ Aqsa Math Art — a custom composition of heart, wave, flower, and sparkle loops  


## 💡 Concept

Every equation starts from zero and builds itself in front of you.  
Because that’s how math actually feels when you truly understand it.


## 🛠️ Built With

- Vanilla JavaScript  
- HTML5 Canvas  
- CSS3 Animations  
- Zero dependencies  


## Why This Project?

This project was built with curiosity, late nights, and a belief that

If you're a student, teacher, developer, or someone who once gave up on math — this is for you.
